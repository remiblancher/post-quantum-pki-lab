# La Revelation - Pourquoi le Post-Quantique ?

**Duree :** 8 minutes
**Prerequis :** Quick Start complete

## Objectif

Comprendre pourquoi la cryptographie post-quantique est urgente et calculer votre propre timeline de migration.

## Contenu

### Partie 1 : Le contexte PQC
- Standards NIST FIPS 2024 (ML-DSA, ML-KEM, SLH-DSA)
- Difference entre cryptographie quantique (QKD) et post-quantique
- Timeline des ordinateurs quantiques

### Partie 2 : La menace SNDL
- Store Now, Decrypt Later
- Qui est concerne (donnees medicales, secrets gouvernementaux, PI)
- Pourquoi agir maintenant

### Partie 3 : Calcul d'urgence
- L'inegalite de Mosca : X + Y > Z = AGIR MAINTENANT
- Calculateur interactif avec vos propres valeurs

## Lancer la demo

```bash
./journey/00-revelation/demo.sh
```

## Ce que vous apprendrez

- Pourquoi RSA et ECDSA sont menaces
- Comment calculer votre urgence de migration
- Les nouveaux standards NIST a adopter
